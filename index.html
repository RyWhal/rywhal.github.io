Object shortcuts

Objects are really straight forward and awesome in coffee-script. You can very dynamically create an object that holds a lot of information. Also, there is no need for all those silly commas in array declarations (if you put each item on its own line).

for example:

user =
    name: "Ryan Whalen"
    username: "RyWhal"
    age: 26

    #No Commas in the array declaration 
    Aliases: [
        'RyWhal'
        'RJPW'
    ]
    websites: [
        'ryanwhalen.me'
        'slur.pe'
        'blackholeoftheinternet.com'
        'rywhal.github.io'
    ]

    #or you can use commas
    emails: ['ryan@ryan.com','whalen@whalen.com']
console.log user

Making a Class

So everything is pretty plain language created in coffee-script. Creating a class is as easy as "class ClasName" and giving it a constructor with some attributes.

uncompiled info_print.coffee

class Info

    constructor: (@attributes)->

    printInfo: ->
        console.log "Name: #{@attributes.name}"
        console.log "Title: #{@attributes.title}"
        console.log "DOB: #{@attributes.dob}"
        if @attributes.salary
            console.log "Salary: #{@attributes.salary}"
        else
            console.log "Salary: Unknown"

ryan = new Info
    name: "Ryan"
    title: "Tech"
    dob: new Date(1988, 2, 2)
    salary: "$1,000,000"

ross = new Info
    name: "Ross S."
    title: "VP"
    dob: new Date(1985, 7, 15)

ryan.printInfo()
console.log " "
ross.printInfo()

after its compiled and run:

$ node info_print.js
Name: Ryan
Title: Tech
DOB: Wed Mar 02 1988 00:00:00 GMT+0000 (UTC)
Salary: $1,000,000

Name: Ross S.
Title: VP
DOB: Thu Aug 15 1985 00:00:00 GMT+0000 (UTC)
Salary: Unknown

Hello World

Turns out a hello world script is pretty easy in coffee script.

hello.coffee:

hello = "Hello, World"
console.log hello

compiled into js:

$ coffee -p hello.coffee
(function() {
  var hello;

  hello = "Hello, World";

  console.log(hello);

}).call(this);

and running it:

$ node hello.js
Hello, World
